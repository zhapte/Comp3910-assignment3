Timesheet REST API – Endpoint List

This page lists all available REST endpoints for Assignment 3.

Authentication

  • POST /api/login — Login and receive token
  • POST /api/logout — Logout (requires token)
  • GET /api/me — Get current authenticated user

User Management (Admin Only)

  • GET /api/users — List all employees
  • POST /api/users — Create new user
  • DELETE /api/users/{username} — Delete user
  • POST /api/users/{username}/reset-password — Reset password
  • PUT /api/users/{username}/password — Update password

Timesheets

  • GET /api/timesheets — Admin: all sheets, User: own sheets
  • GET /api/timesheets/{id} — Load specific timesheet
  • POST /api/timesheets — Create current week’s timesheet
  • PUT /api/timesheets/{id} — Update timesheet

Notes

  • All secured endpoints require header:
    Authorization: Bearer <token>
  • Use Postman or curl for testing.
